# bull-bullmq-nestjs


## Description
Patrones de colas de tareas asíncronas con Bull/BullMQ en NestJS para producción. Usar PROACTIVAMENTE cuando se trabaje con jobs programados, recordatorios, notificaciones asíncronas, procesamiento en background, o cualquier tarea que no deba ejecutarse de forma síncrona en el ciclo de request/response. Activar siempre que aparezcan las palabras clave: Bull, BullMQ, @BullModule, @Processor, @Process, @OnQueueFailed, addJob, Queue, Worker, jobs, colas, tareas programadas, delayed jobs, recordatorio programado, notificación asíncrona, background job, retry, dead letter queue, o colas con Redis en NestJS. También activar para la transición Bull (desarrollo) → Cloud Tasks (producción GCP).
