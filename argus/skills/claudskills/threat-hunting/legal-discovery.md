# legal-discovery


## Description
Audit e-discovery and litigation document review systems -- data collection pipelines (PST, MBOX, SharePoint, Slack), document processing (OCR via Tesseract/ABBYY, metadata extraction, deduplication), Technology Assisted Review (TAR 1.0/2.0/CAL with recall/precision tracking), privilege detection (attorney roster matching, keyword flagging, privilege log generation), PII redaction (SSN, credit card, HIPAA data with irreversible burn-in), production formatting (TIFF/PDF, Bates numbering, Concordance/Relativity load files), and defensibility audit trails (chain of custody, hash verification). Use when reviewing legal tech platforms, document review tools, or any codebase handling FRCP litigation holds, EDRM workflows, or court production sets.
