# datasheet-key-extractor


## Description
Extrae parámetros críticos de datasheets técnicos de componentes electrónicos y microcontroladores, y los resume en lenguaje natural accesible para estudiantes. Activa cuando el usuario menciona: datasheet, hoja de datos, parámetros técnicos, especificaciones, extraer datos, datasheet extraction, technical specs, electrical characteristics, pinout, absolute maximum ratings, o cuando se necesita entender las capacidades y limitaciones de un componente electrónico para contenido educativo. Usar SIEMPRE después de document-structure-mapper cuando se detecten archivos PDF de datasheets.
