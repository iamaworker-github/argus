# peripheral-code-wizard


## Description
Genera fragmentos de código inicial funcionales para Arduino IDE y ESP-IDF a partir del mapeo de pines y periféricos de un microcontrolador ESP32. Activa cuando el usuario menciona: código, code, programar ESP32, Arduino, ESP-IDF, GPIO, periférico, peripheral, firmware, sketch, ejemplo de código, pin mapping, o cuando se necesita generar código que controle un periférico específico del microcontrolador para un proyecto de domótica. Usar SIEMPRE después de schematic-storyteller para traducir las conexiones de hardware en código ejecutable.
