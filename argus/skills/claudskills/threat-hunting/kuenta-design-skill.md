# kuenta-design


## Description
Design system y guía visual completa para el proyecto KUENTA (PWA de pagos P2P Chile). Usar SIEMPRE que se cree o modifique cualquier componente visual, pantalla, estilo, layout o elemento de interfaz en el proyecto KUENTA. Incluye tipografía (Syne + DM Sans), paleta de colores con tokens CSS, glassmorphism, wordmark, modos claro/oscuro, iconografía SVG, animaciones, componentes y checklist de implementación. También usar cuando se mencione: wordmark, splash screen, login, header, tab bar, cards, botones, inputs, avatares, colores, fuentes o cualquier elemento visual de KUENTA.
