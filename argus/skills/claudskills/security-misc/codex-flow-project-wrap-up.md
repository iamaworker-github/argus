# codex-flow-project-wrap-up


## Description
Użyj, gdy kończysz sesję pracy nad projektem i chcesz uporządkować stan dokumentacji operacyjnej bez implementacji nowych zmian: zaktualizować ROADMAP.md, STATUS.md oraz w razie potrzeby README.md, a następnie upewnić się, że bieżący stan jest zapisany w repo. Ten skill nie służy do wdrażania funkcji ani do tworzenia nowego planu.
