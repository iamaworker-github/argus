# neo4j-aura-provisioning-skill


## Description
Use when programmatically creating, pausing, resuming, resizing, or deleting Neo4j Aura instances: Aura CLI (aura-cli), Aura REST API, or scripted provisioning in CI/CD pipelines. Also covers downloading credentials, listing tenants, and managing instance lifecycles. Does NOT cover Cypher queries or application code — use neo4j-cypher-skill or driver skills. Does NOT cover general CLI tools (neo4j-admin, cypher-shell) — use neo4j-cli-tools-skill.
