# context-checkpoint


## Description
Save, compress, version, encrypt, and resume full chat context when approaching token/message limits. Use when the conversation is long, the user says save our progress or context is running out, approaching token limits, resuming a session across chat windows or different AI models, creating named bookmarks, merging parallel chats, or exporting conversation history as knowledge assets. Supports commands like /save, /resume, /resume light, /bookmark, /list bookmarks, /recover_last, /merge, /diff, /export.
