# x402-payments


## Description
Use whenever the user wants to add payments to an API, monetize endpoints, implement x402, handle HTTP 402 responses, create paid APIs, set up crypto paywalls, accept USDC, or build agent payments. Trigger on: x402, 402 Payment Required, paid API, paywall, micropayments, USDC payments, agent payments, monetize API, payment middleware, x402-next, x402-express, x402-fetch, Sentinel, payment router, audit trail, budget caps, payment receipts, facilitator. Also trigger when turning a free API into paid, adding per-request pricing, or building AI agent autonomous payment flows. Covers Base (EVM) and Solana with USDC. Covers Chainlink CRE (Chainlink Runtime Environment) workflows triggered by x402 payments. Trigger also on: Chainlink CRE, CRE workflow, onchain automation with payments, price alerts with x402, Chainlink price feeds with payments, x402 CRE integration, paid CRE workflows, monetize Chainlink workflows. Even without "x402" mentioned — if they want to charge for API access with crypto or build agent payment infrastructure, this skill applies.
