# bmad-nfr-assess


## Description
Assesses Non-Functional Requirements (security, performance, reliability, maintainability, observability) with evidence-based codebase analysis. Scans actual code for patterns, scores each aspect, and produces a structured assessment saved as a tracker document. Supports epic-level (specific project) or system-level (whole codebase) scope.
